The program has the ability to run in two modes, one with the provided height map only, and with a colour map for the height map. If the colour map cannot be found, it will default to a height map only operation 

CONTROLS:

ESCAPE: 		closes the window
UP:			translates map up
LEFT:			translates map left
DOWN:			translates map down
RIGHT:			translates map right
Z:			zoom in
X:			zoom out
I:			rotates the map around the x-axis, counter-clockwise
K:			rotates the map around the x-axis, clockwise
J:			rotates the map around the y-axis, counter-clockwise
L:			rotates the map around the y-axis, clockwise
W:			renders map using lines 
P:			renders map using points 
T:			renders map using triangles
